simplegame — Godot 4 office walkaround

Flow: startup (Matter) → login → main game

Controls
  W A S D — move
  E — interact (NPCs, computers, terminals)
  Esc — close popups / telephone chat

Folders
  assets/   — sprites and data files
  scenes/   — startup, login, main
  scripts/  — game logic

Open project.godot in Godot 4, press F5.

Project folder should be named: simplegame
