All game assets for this project are stored in this folder.

Files
-----
16x16 Walk-Sheet.png
  Player walk animations (5 rows x 4 frames, 24x24 px each).
  Used by the player and NPC sprites.

16x16 Run-Sheet.png
  Run animation sheet (not used in the game yet).

example.http
  Sample HTTP text file shown on the office computer (press E).

Godot paths
-----------
Reference assets from code or scenes using:

  res://assets/16x16 Walk-Sheet.png
  res://assets/16x16 Run-Sheet.png
  res://assets/example.http

Add new art or data files here and use the same res://assets/... prefix.
